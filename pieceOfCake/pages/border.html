Border

Piece of Cake has a basic set of colors for text, links, and backgrounds. These colors help to create uniformity and harmony in the look of UI elements. All colors are carefully selected and combined with each other. Of course, you can change the color scheme to your choice in the framework settings.

  • Border Colors

Adding Borders

The poc-border classes are used to add borders to any HTML element:

.poc-border-0
.poc-border
.poc-border-bottom
.poc-border-right
.poc-border-left
.poc-border-top

Border Colors

The poc-border-color classes define the border color for any HTML element:

.poc-border-blue
.poc-border-red

Code

   	                            <div class="poc-border-red"></div>

Rounded Borders

To add rounded borders, add one of the poc-round-size classes:

.poc-border .poc-round-small
.poc-border .poc-round-medium
.poc-border .poc-round-large
.poc-border .poc-round-xlarge
