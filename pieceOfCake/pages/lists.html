Lists

Piece of Cake has most basic list group is an unordered list with list items and the proper classes. Build upon it with the options that follow, or with your own CSS as needed.

  • Basic Lists
  • Bordered Lists
  • Rounded Borders

Basic Lists

The poc-ul class is used to display a basic list:

  • Volkswagen
  • Ford
  • Toyota
  • Honda

Code

   	                            <ul class="poc-ul">
   	                                <li>Volkswagen</li>
   	                                <li>Ford</li>
   	                                <li>Toyota</li>
   	                                <li>Honda</li>
   	                            </ul>

Border Colors

Combined poc-ul and poc-border classes will define a list with borders:

  • Volkswagen
  • Ford
  • Toyota
  • Honda

Code

   	                            <ul class="poc-ul poc-border">
   	                                <li>Volkswagen</li>
   	                                <li>Ford</li>
   	                                <li>Toyota</li>
   	                                <li>Honda</li>
   	                            </ul>

Colored Lists

To add colors, add one of the poc-color classes:

  • Volkswagen
  • Ford
  • Toyota
  • Honda

Code

   	                            <ul class="poc-ul poc-border">
   	                                <li class="poc-pale-green>Volkswagen</li>
   	                                <li>Ford</li>
   	                                <li class="poc-pale-green>Toyota</li>
   	                                <li>Honda</li>
   	                            </ul>
