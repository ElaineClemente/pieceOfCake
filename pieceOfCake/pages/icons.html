Icons

With Piece of Cake you can use the icon library you like, such as Font Awesome Icons, Google Material Design Icons or Bootstrap Icons.

Using an Icon Library

To insert an icon first, include the icon library from a CDN (Content Delivery Network) in the <head> section.

Second, add the name of the icon class to any inline HTML element.

Code

   	                            <div class="poc-xlarge poc-text-blue">
   	                                <i class="fa fa-home></i>
   	                                <i class="fa fa-search></i>
   	                                <i class="fa fa-trash></i>
   	                                <i class="fa fa-at></i>
   	                                <i class="fa fa-calendar></i>
   	                                <i class="fa fa-mobile-at></i>
   	                            </div>

Controling Size

To control the size of the icon, change the font-size property of the icon, or use one of the poc-size classes:

  • poc-tiny
  • poc-small
  • poc-medium
  • poc-large
  • poc-xxlarge
  • poc-xxxlarge
  • poc-jumbo

Code

   	                            <ul class="poc-text-blue">
   	                                <li class="poc-xlarge"><i class="fa fa-home></i>Home</li>
   	                            </ul>
