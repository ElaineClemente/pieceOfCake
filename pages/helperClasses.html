Helper Classes

POC provides a few additional classes that can be helpful in certain cases.

Quick Float Left

Add class="poc-fl-l" to any element to float it to the left. For example:

Code

        <div class="poc-fl-l"></div>

Quick Float Right

Add class="poc-fl-r" to any element to float it to the right. For example:

Code

        <div class="poc-fl-r"></div>

Clearfix

If you need to clear floated elements all you need to include is an empty div with the ".poc-clearfix" class after the last floated item.

Code

        <div class="poc-clearfix"></div>

No Print - Helper

If you want elements like images to be hidden when printing your website, just add the ".poc-noPrint" class.

Code

        <div class="poc-noPrint"></div>
